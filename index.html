JMonkeyRunner

Overview

JMonkeyRunner is a Java implementation of Androids monkeyrunner.

Running

Please have a look at the jmonkeyrunnerexamples on how to use this library.

Development

To use this code you will need the following jars from the Android SDK folder:

  • common.jar
  • ddmlib.jar
  • quava-15.0.jar

as well as:

  • chattychimpchat (latest version)
